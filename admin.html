⚙️ One-Time Firebase Setup — Follow These Steps

  1. Go to console.firebase.google.com → Create a new project (it's free).
  2. Inside your project → click Firestore Database → Create database → select Start in test mode → click Next → Enable.
  3. Click the ⚙️ gear icon → Project settings → scroll to Your apps → click </> Web → Register the app → copy the firebaseConfig object shown.
  4. Open both admin.html and index.html in Notepad/VS Code → find the line PASTE YOUR FIREBASE CONFIG HERE → replace the placeholder config with your real one.
  5. Save both files. Now songs added here will instantly appear for every visitor on any device! 🎉
